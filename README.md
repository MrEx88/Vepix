# Vepix
#### A picture viewer/editor application

#### Capabilities so far:
* Edit picture name(s)
* Delete picture
* move picture
* copy picture

* Has command line switches:

	> -f "C:\Users\Me\Pictures\2016", "C:\Users\Me\Pictures\2015"
	
	> -t "C:\Users\Me\Pictures", "H:\Backup\Pictures"
	
	> -p "\*.jpg", "\*2\*.png"
